# 🚀 DevDesk

> **Build. Learn. Grow.**

DevDesk is a modern all-in-one productivity web application designed to help students and developers organize their daily workflow, improve focus, and monitor productivity through real-time analytics.

---

## ✨ Features

### 📝 Notes
- Add, edit and delete notes
- Duplicate note prevention
- Local Storage support

### ✅ Task Manager
- Add tasks with priority & due dates
- Complete / Delete tasks
- Search tasks
- Sort tasks
- Filter completed & pending tasks

### 🎯 Today's Focus
- Set daily focus goal
- Personalized productivity tracking

### 🍅 Pomodoro Timer
- Fully functional Pomodoro timer
- Custom timer duration
- Start / Pause / Reset
- Completion notification
- XP rewards

### 📅 Planner
- Daily planning system
- Track upcoming tasks

### 🤖 AI Productivity Coach
- Personalized productivity tips
- Dynamic recommendations

### ⚡ AI Smart Schedule
- Automatically generates daily schedule
- Based on productivity data

### 📊 Productivity Dashboard
- Notes Counter
- Focus Counter
- Pomodoro Counter
- Goal Completion
- XP System
- Productivity Level

### 📈 Advanced Analytics
- Overall Productivity Score
- Weekly Productivity Trend
- Task Distribution Chart
- Priority Distribution Chart
- Completion Percentage
- Monthly Activity Heatmap
- Best Productivity Day
- Last 7 Days Summary

### 🏆 Gamification
- XP System
- Levels
- Achievements
- Productivity Rating
- Streak Tracking

### 🌤 Extra Features
- Weather Widget
- Daily Motivational Quotes
- Notifications
- PDF Export
- Backup & Restore
- Dark Mode
- Settings
